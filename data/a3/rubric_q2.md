# Marking Guide for Q2 (5 marks)

| Tasks                                          | Marks | Topic         | Why?                       |
| ---------------------------------------------- | ----- | ------------- | -------------------------- |
| Generating random number (1-100)               | +1    | Randomness    | Setting up the game state  |
| Loop for guessing (while/do-while)             | +1    | Loops         | Allowing multiple attempts |
| Logic for "Too high" hint                      | +1    | Conditionals  | Guiding the user           |
| Logic for "Too low" hint                       | +1    | Conditionals  | Guiding the user           |
| specific termination condition (correct guess) | +1    | Boolean Logic | Ending the game correctly  |
